Mahindra XUV 3XO

Features Report.

General Features

The Mahindra XUV 3XO is a sub-4 meter SUV available with a range of engine options:

  • 1.2-litre Turbo-Petrol: 111 bhp, 200 Nm torque, 6-speed manual/automatic.
  • 1.2-litre TGDi mStallion Turbo-Petrol: 128 bhp, 230 Nm torque, 6-speed manual/automatic.
  • 1.5-litre mHawk Diesel: 115 bhp, 300 Nm torque, 6-speed manual/AMT.

Dimensions:

  • Length: 3990 mm
  • Width: 1821 mm
  • Height: 1647 mm
  • Wheelbase: 2600 mm
  • Boot Space: 364 Litres

Key features:

  • 10.25-inch touchscreen infotainment system with Android Auto and Apple CarPlay.
  • 10.25-inch digital instrument cluster.
  • Connected car technology.

Luxury Features

The Mahindra XUV 3XO includes several luxury features designed to enhance comfort and convenience:

  • Panoramic Sunroof (largest in segment)
  • Premium Leatherette Upholstery
  • Automatic Climate Control (Dual-Zone)
  • Harman Kardon 7-speaker audio system
  • Ventilated Seats
  • Cooled Glovebox

Safety Features

The Mahindra XUV 3XO places a strong emphasis on safety with a comprehensive suite of features:

  • 6 Airbags (Standard)
  • ABS with EBD
  • ESC
  • TPMS
  • 360-Degree Camera
  • ADAS Level 2:
    • Adaptive Cruise Control
    • Autonomous Emergency Braking
    • Lane Keep Assist
    • Forward Collision Warning

The Mahindra XUV 3XO offers a compelling value proposition in the sub-4 meter SUV segment, combining a comprehensive feature set, a strong emphasis on safety, and a stylish design. The inclusion of ADAS Level 2 features gives it a distinct advantage over many competitors.
