Powertrain Options

The Mahindra XUV 3XO is available with multiple engine options:

  • 1.2-litre turbo-petrol engine (111hp, 200Nm)

Transmission options include a 6-speed manual gearbox and a 6-speed torque converter automatic gearbox (for petrol engines), and a 6-speed AMT unit (for diesel engines).
